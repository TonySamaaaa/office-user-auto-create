## 🚚 编译

1. 安装软件包

    + `apt install git g++ make python2`

2. 克隆仓库

    + `git clone -b static https://github.com/TonySamaaaa/office-user-auto-create.git`

3. 编译

    + `yarn`
    + `yarn build`
